# Telemetry log reader

To start a project you need .net 5+

You can download this here (https://dotnet.microsoft.com/en-us/download)

## To start

dotnet build

dotnet run --project sw_course <i>input-file</i> <i><output-file</i>

## To start test

dotnet test
